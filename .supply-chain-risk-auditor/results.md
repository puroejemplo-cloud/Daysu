## Suggested Alternatives

- **`next` 16.2.6**: Actualización de parche, completamente compatible. Corrige PostCSS XSS y otros CVEs moderados. Ejecutar `npm install next@16.2.6`.
- **`@vercel/blob` 2.3.3**: Major version bump (0.x → 2.x). Revisar si la API cambió. Corrige 5 CVEs en undici incluyendo 2 HIGH (DoS via WebSocket).
- **`argon2`**: Reemplaza bcryptjs con hashing Argon2id, ganador de la Password Hashing Competition. Más resistente a ataques de GPU/ASIC. Requiere Node.js nativo (no funciona en Edge runtime). Si se mantiene bcryptjs, asegurarse de que el lockfile esté commiteado y la versión exactamente pinned.
- **Eliminar ffmpeg-static + ffprobe-static + fluent-ffmpeg**: Si solo se usan en scripts de procesamiento de galería (que corren en máquinas dev o CI), evaluar reemplazarlos por una instalación del sistema (`brew install ffmpeg`, Docker con ffmpeg, o scripts de CI que instalen ffmpeg directamente). Esto elimina el vector de binarios comprometidos.
